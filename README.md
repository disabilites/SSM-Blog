# 基于SSM框架实现的简易博客系统

### 开发环境
- Inteilj IDEA
- Maven
- Tomcat9
- JDK11

---

### 实现的功能
1. 文章归档
2. 文章分类
3. 文章留言
4. 图片上传
5. markdown编辑
